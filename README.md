# HubSpot API Documentation (AI-ready)

This repository contains cleaned, structured HubSpot API documentation
converted to Markdown for automated workflows and AI agents.

## Structure

- docs/        → HubSpot documentation in Markdown
- indexes/     → Machine-readable indexes (used by AI agents)
- scripts/     → Utilities to build and validate indexes

## Source

Docs are derived from:
https://developers.hubspot.com/docs/
